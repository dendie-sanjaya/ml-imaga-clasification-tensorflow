Untuk Menjalankan API Service
 python -m venv env  (untuk masuk ke enviroment)
 env\Scripts\activate or env/bin/activate (untuk masuk ke enviroment windows )
 source env\Scripts\activate or env/bin/activate (untuk masuk ke enviroment linux)
 
 python app.py
 nohop python app.py (untuk menjalan service di background atau menjadi daemon)
